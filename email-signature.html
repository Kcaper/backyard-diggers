How to load this in Gmail

1. In this page, select the whole signature in the white box below (click just before the logo, drag to the end of the last line).
2. Copy it (⌘C).
3. Gmail → Settings ⚙ → See all settings → General → Signature → Create new.
4. Paste (⌘V) into the signature box, set it as default for new mail & replies, then Save Changes at the bottom.
